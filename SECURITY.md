# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this package, please report it responsibly. **Do not open a public GitHub issue.**

Email **oss@hackthebox.com** with:

- A description of the vulnerability
- Steps to reproduce
- Affected versions
- Any potential impact you have identified

We will acknowledge your report within 5 business days and aim to provide a fix or mitigation plan within 30 days, depending on severity.

## Supported Versions

Security fixes are applied to the latest minor release only.

| Version | Supported |
|---------|-----------|
| 1.x     | Yes       |
